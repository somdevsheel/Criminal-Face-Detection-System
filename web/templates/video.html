Instructions

Webcam Detection:

  • Click "Start Webcam" to begin live detection
  • Matched faces will be automatically saved
  • Check database for recorded matches
  • Press 'Q' in video window to quit (if running locally)

Video File Processing:

  • Upload CCTV footage or video file
  • System will process and detect faces
  • Results will be saved with annotations
  • Check logs for match details

For Local Testing (Command Line):

# Webcam
python src/video_processor.py --mode webcam

# Video file
python src/video_processor.py --mode video --source path/to/video.mp4

# RTSP stream (IP camera)
python src/video_processor.py --mode rtsp --source rtsp://camera_url
